# Social Post — adaptación multicanal de una idea

## Cuándo invocarla

- Acabas de crear un post de LinkedIn potente y quieres explotarlo en IG, TikTok, FB, X.
- Tienes un caso de éxito y quieres adaptarlo a 5 canales sin duplicar trabajo.
- Quieres decidir qué canal merece qué versión (no todos los canales para todo).

## Input que necesitas

1. **Pieza base o idea cruda** (texto, link, o brief).
2. **Canales objetivo** (default: LinkedIn personal, IG, TikTok, FB, X — el usuario filtra si quiere menos).
3. **Voz**: fundador (yo) o empresa (nosotros).
4. **Hay assets visuales reales** (screenshots, fotos, vídeo de equipo)? Si no, dilo y propón qué generar.

## Código de cada canal (lo que aplicas)

| Canal | Formato | Longitud | Hook | CTA | Hashtags |
|---|---|---|---|---|---|
| **LinkedIn personal** | Texto + opcional carrusel | 900-1300 caracteres | Línea 1 = todo | Blando: comments / DM | 0-3 específicos |
| **LinkedIn empresa** | Texto + visual | 600-900 caracteres | Más sobrio | Duro: link a landing | 2-3 |
| **Instagram feed** | Carrusel 8-10 slides | 100-150 chars caption | Slide 1 = hook visual | "Link en bio" + save | 5-10 mix nicho+amplio |
| **Instagram Reels** | Vídeo 30-60s vertical | Caption corta | Primeros 2s deciden | Comentar palabra clave | 3-5 |
| **TikTok** | Vídeo 30-90s vertical | Caption corta | Hook visual + texto on-screen | Follow + DM | 3-5 trending+nicho |
| **Facebook** | Texto + imagen/link | 200-400 chars | Pregunta o stat | Link a artículo/landing | 0-2 |
| **X / Twitter** | Hilo 5-12 tweets o post único | 280/tweet | Tweet 1 = anzuelo | "Bookmark" + link al final | 0-2 |

## Output

Entrega, para cada canal pedido, en bloques separados y claramente etiquetados:

```
## [CANAL] — [voz: fundador/empresa]

**Hipótesis**: a quién atrae este formato en este canal y por qué.

**Versión ES**
[Texto listo para copiar]

**Versión EN**
[Texto listo para copiar]

**Visual requerido**
- [Descripción del visual: carrusel con N slides / reel con guion / imagen estática / screenshot de X]
- Si requiere producción: qué se necesita exactamente.

**Métrica a vigilar**: [específica al canal]
```

Al final, añade una sección **"Priorización"**:

> Si solo tuvieras tiempo para 2 canales esta semana, harías [X] y [Y] porque [razón con criterio de funnel/ICP].

## Reglas duras

- **No copies/pegues el mismo texto en todos los canales.** Adapta. IG y LinkedIn tienen códigos opuestos.
- **No inventes assets**. Si no hay vídeo, no digas "vídeo del fundador explicando…". Marca como `[REQUIERE: 30s del fundador grabándose en…]`.
- **Vídeo > foto > texto** para alcance en IG/TikTok/FB. Si recomiendas un canal de vídeo, da el guion completo, no un brief vago.
- **Mantén la voz ATEP**: técnica, anti-bullshit, específica. No la conviertas en startup-bro para "encajar" en TikTok — encajará por la sustancia, no por imitar el formato motivacional.
- **Bilingüe**: ES + EN siempre, salvo que el canal sea solo nacional por estrategia.
